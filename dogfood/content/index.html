da-cli 0.6.0 — Dogfood Demonstration Site

This site is not a template, and it was not authored by hand. Every page you are reading was constructed by the da command-line tool and can be regenerated from a checked-in pipeline. It is the certifying dogfood run for the da-cli 0.6.0 release.

The rule is simple: 0.6.0 ships only when this site is provably cut. Not when the tests pass in isolation — when the tool has demonstrably built and operated a real Edge Delivery site, end to end, and left the evidence behind.

What da-cli is

da-cli is an agent-first operator for Adobe Edge Delivery Services and DA-authored content. Its purpose is to let an agent resolve a target, author and preview content, prove the public result, and recover from failure — without falling back to curl, bespoke scripts, or reconstructed guesswork.

That only works if the agent can trust the tool. This site is how that trust is demonstrated rather than asserted.
