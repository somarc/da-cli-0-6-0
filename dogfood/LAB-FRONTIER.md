# Lab: Frontier — living platform proof

**Route:** `/frontier` · **Kind:** Lab (CONTENT-IA) · **Blocks:** `frontier-hero`, `scroll-rail`, `vf-specimen`, `feature-matrix`

## Provenance

Study reference: `https://jovial-kayak-tysa.here.now/` — *FRONTIER/26 — The State of
the Web Platform, July 2026*, generated by the K3 frontier model from the prompt
"push the boundaries of what's possible in web design with the absolute latest in
css, webgl, shaders … create the most impressive, informational single-page
experience you can imagine."

Teardown facts (verified 2026-02-27):

- One self-contained HTML file, ~66 KB: ~30 KB CSS, ~13 KB JS, no frameworks,
  no build, no images or video. Only external dependency: Google Fonts.
- Hero: raymarched gyroid SDF in a ~120-line GLSL ES 3.00 fragment shader,
  fullscreen triangle via `gl_VertexID` (no vertex buffers), cosine palette,
  fresnel, exponential fog.
- Chapter 02: 420vh wrapper + named `view-timeline` drives a horizontal card
  rail — zero JS, compositor-run. Progress hairline via
  `animation-timeline: scroll(root)`. Reveals via `view()` +
  `animation-range: entry 0% entry 55%`.
- Whole page themed by ONE registered `@property --hue`; every accent is
  `oklch(L C calc(var(--hue) ± n))`; `color-mix(in oklch, …)` ladders.
- Chapter 06: a live self-audit — 24 `CSS.supports()`/API probes rendered as a
  pass/fail grid. "Every green dot is a feature live on this page."
- PE discipline: reduced-motion skips shader boot entirely (static fallback
  div); WebGL failure → same fallback; no view timelines → IntersectionObserver
  class; no `startViewTransition` → instant swap.
- Perf: DPR capped `min(dpr, 1.5) × 0.7`, IO start/stop per canvas, lerped
  pointer uniforms, `antialias: false`.

## Design intent (the remap)

FRONTIER/26 is a dark-void neon dossier. This site is Renderaissance paper.
We do **not** clone the void. The remap: each frontier demo becomes a
**dark instrument plate** — a framed, rounded dark stage sitting on the paper
field, like a lab specimen under glass. Copy stays in site type
(Cormorant/Alegreya/Jost); the plates carry the OKLCH accent system.

One shared knob maps the **brand triad** onto a single number:
`--frontier-hue` (registered `@property`, default `40` = terracotta).
Chartreuse = `hue + 80` (≈120), cobalt ink = `hue + 225` (≈265). The shader
uses the same anchors: a linear-light ramp ink → cobalt → terracotta → peach,
with a peach/chartreuse fresnel rim — the raymarcher is lit in Renderaissance.

Thesis line: **the page proves its own claims** — the same ethos as this
site's "living documentation that certifies the CLI surface." A lab page
where every capability statement is executable evidence.

## Hard EDS rules (inherited from Dualform, adapted)

1. **LCP is a static image** — the hero poster (`media/frontier/*.webp`).
   The canvas boots after first paint and fades in over the poster.
2. **Static path complete** — with JS off the page reads fully: poster hero,
   vertically stacked rail cards, probe list as plain text.
3. **Effects are progressive; fail soft** — no WebGL2 → poster stays; no
   scroll timelines → static rail; every probe failure renders as data,
   not breakage.
4. **`prefers-reduced-motion`** — shader never boots; rail never slides.
5. **Authors edit content, not shaders** — DA = copy + poster image +
   plain config rows (`hue`, `preset`). Shaders live in block code.
6. **No scroll-jacking** — the rail is honest scroll (sticky + timeline),
   capped in height, skippable.
7. **CDD** — fixture first (`dogfood/fixtures/frontier.html`), lint,
   preview URL before any publish. No live publish without explicit ask.

## Block contracts

### `frontier-hero`
Author shape: one row, two cells — copy (eyebrow p / h1 / lede p) | poster img.
Optional config rows after: `hue | <0–360>`, `preset | gyroid`.
Decoration builds a plate stage: poster `<picture>` (LCP) + overlay canvas
(opacity 0 → 1 on first rendered frame) + HUD spec chips. IO starts/stops the
render loop. Pointer lerps a warp uniform.

### `scroll-rail`
Author shape: each row = one card, one cell (index p / h3 / body p).
Decoration wraps cards in a sticky stage + horizontal track; a named
`view-timeline` on the block maps vertical scroll to horizontal translate;
a progress hairline fills on the same timeline. Engines without
`animation-timeline: view()` (or reduced motion) get class `sr-static`:
plain vertical cards.

### `vf-specimen`
Author shape: row 1 = specimen text; optional rows `voice | <name>`,
`font | fraunces`. Axis registry, named voices, and font-CSS injection live
in block JS — the font stylesheet loads only when the block decorates, so
the global LCP path never pays for it. Specimen is contenteditable; sliders
drive `font-variation-settings`; voice jumps morph via CSS transition
(disabled under reduced motion). JS off = specimen in the site serif.

### `feature-matrix`
Author shape: zero rows = full curated probe registry; optional rows filter
by kind (`css` / `js`). Registry lives in block JS (authors never write
probes). Renders pass/fail grid + score sentence. With JS off the block
shows its authored caption text only.
