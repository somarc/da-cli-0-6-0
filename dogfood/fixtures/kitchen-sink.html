Why a kitchen sink

Wave 1 proved the author → preview → verify loop. Wave 2 proves the block, contract, and design surfaces. A page with only columns and cards cannot certify those surfaces. This page carries one instance of each module in the modern-site MVP spectrum.

FAQ (accordion)

What makes Wave 2 different from Wave 1?

Wave 1 is the content CRUD and delivery loop. Wave 2 requires real block decoration, code-bus assets, and audit/design surfaces to run against non-trivial structure.

Why not only use dualform and painterly heroes?

Aesthetic PE is necessary but not sufficient. A release dogfood must exercise the boring modules too — tables, tabs, accordions — because production sites are full-spectrum.

How do we know a block is proven?

da audit contracts --verify-code returns status ok for the block’s JS and CSS on the served branch, and da audit full reports no structural errors on this page.
