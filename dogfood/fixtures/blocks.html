Authoring versus proving

Authoring

Blocks are authored as tables — a class name and nested rows and cells. da content put uploads the source verbatim; Edge Delivery decorates it into the rendered block.

Proving

Uploading source is not proof. da audit contracts --verify-code confirms the block's code exists on the codebus branch preview renders from. A block that authors cleanly but has no code on the branch is caught here, not by a user.
